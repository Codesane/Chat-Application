The Database

Communicates With
The database is somewhere in the middle of the server and the website. When a new user is registered (s)he is instantly added to the database. When the user later tries to login to the server using the client chat application, the application will send a request to the server asking to query the database for the account information concerning the user, to which the server replies with only a small amount of the data to client (username and id).

Connectivity
The connection between the database and the Server is maintained by a JDBC Driver, the website uses PHP to connect to the database.
